# Garage Project

## Overview
This repository contains the source code and documentation for the Garage project, which is aimed at providing a seamless experience for managing and utilizing garage spaces.

## Features
- **User Management**: Register and manage user profiles.
- **Space Listings**: View and filter available garage spaces.
- **Booking System**: Book garage spaces with real-time availability.
- **Payment Integration**: Secure online payments for bookings.

## Tech Stack
- **Frontend**: React, HTML, CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/sujithnirmal/garage.git
   ```
2. Navigate to the project directory:
   ```bash
   cd garage
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
